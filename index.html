Privacy Policy
We're using the Dropbox API, Google Analytics and the Facebook Share Button.

Dropbox API

We're using the Dropbox API (by using this JavaScript library): dropbox-js. To visualize your folders and files, we need access on your whole Dropbox. For the functioning of our app, we would only need Read access - unfortunately Dropbox does support only a combined Read/Write access. We do not make use of this Write permission. In fact, we do not store a single bit of your Drobpox information when visualizing your Dropbox folder (and we do not plan to do so). If you don't trust us, take a look at our website's source code on Github.

Google Analytics

As many sites around the web, we use Google Analytics to keep track about the number of visitors we have. Google Analytics places a cookie on your computer to identify you on later visits. To opt out, visit one of Google Analytics currently available opt-outs.

Facebook Share Button

Our website is using Social Plugins of facebook.com. This means, every time you are using our website, a connection to Facebook is created. We can not control how Facebook uses this data. It could be used to identify you when visiting other websites using the Social Plugins or when using the Facebook website. You can find the Facebook privacy policy here: http://www.facebook.com/policy.php.
